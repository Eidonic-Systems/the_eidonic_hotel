# 🏨 Eidonic Hotel

> *A surreal cosmic horror point & click adventure*  
> *A place that should not exist… yet you’ve already checked in.*

---

## 🧠 Overview

**Eidonic Hotel** is a fully original, AI-assisted, narrative-driven point & click adventure built from scratch in Godot.

Every system, visual, interaction, and mechanic has been independently designed and developed using a hybrid workflow combining generative AI, manual refinement, and custom scripting.

This project represents the intersection of:
- Generative AI image creation
- Interactive system design
- Environmental storytelling
- Experimental game development

---

## 🎮 Gameplay

Eidonic Hotel follows a **classic verb-based point & click system**, inspired by SCUMM-style interactions.

### Core Mechanics:
- 🔍 **Verb System** – Look, Use, Push, Pull, Open, Close, Talk, Eat  
- 🎒 **Inventory System** – Item-based puzzle solving  
- 🧩 **Multi-step puzzles** – Cause → effect → transformation  
- 🧠 **Environmental storytelling** – Rooms evolve as you interact  
- 🎭 **Dynamic interactions** – Objects and characters react, change, and sometimes… remember  

Each room functions as a contained narrative puzzle space with unique mechanics and visual transformations.

## 🧰 Development Approach

This project was built entirely from scratch with no external assets.

### Visual Pipeline:
- AI-generated imagery (multi-model workflows)
- Manual refinement and compositing in Photoshop
- Optimized for in-engine integration

### Systems & Code:
- Custom-built gameplay systems (Godot / GDScript)
- Modular event sequencing
- State-driven interactions and persistence systems

### Design Philosophy:
- Build first, refine through iteration  
- Treat visuals and mechanics as one system  
- Embrace unpredictability as part of the experience
